# 🚀 Konecta Partner - PWA

Plataforma completa para franqueados Konecta.

## 📋 Funcionalidades

### Painel da Matriz
- Cadastrar/Editar franqueados
- Ativar, inativar, bloquear
- Gerar link personalizado
- Copiar link
- Visualizar app do franqueado
- Definir URL base
- Contadores (total, ativos, inativos, bloqueados)

### App do Franqueado
- Visualização de dados pessoais
- Cadastro de clientes
- Geração de propostas com cálculos automáticos
- Materiais de apoio
- Checklist de implantação

## 🛠 Tecnologias
- HTML5 + CSS3 (Design System Premium)
- JavaScript Vanilla
- PWA (Service Worker + Manifest)
- localStorage (dados locais)
